# プロプライエタリソフトウェア

## 説明

プロプライエタリソフトウェアとは、著作権などの権利が明確に所有者に帰属しており、その所有者が利用者に対して自由に使用・複製・改変・再配布などの取り扱いを制限・管理しているソフトウェアのことを指します。
具体的には、ソースコードを一般に公表せず、バイナリコードのみを提供する形態で、利用許諾契約によって複数のコンピュータへの導入を禁じたり、起動前位に認証手続きを義務付けたりすることもあります。
ただ、ソフトウェア製品の中にも、企業開発のLinuxディストリビューションのようにオープンソースとして公開されているものもあれば、利用者側に一定の条件のもとソースコードを提供する製品もあるため、「商用ソフトウェア」とも「クローズドソースソフトウェア」とも必ずしも一致しない。
