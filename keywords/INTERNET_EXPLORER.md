# Internet Explorer

## 説明

マイクロソフトがかつて開発していたWebブラウザで、Windows 95からWindows 10までに標準で含まれている。Windows 10から標準ブラウザはMicrosoft Edgeに置き換えられたことでInternet Explorerの開発は終了し、マイクロソフトはInternet Explorerを利用しないことを呼びかけている。Windows 11にはInternet Explorerは搭載されていない。
