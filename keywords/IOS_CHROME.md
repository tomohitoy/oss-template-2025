# iOS版 Chrome

## 説明

iOS版Chromeとは、Googleが提供するWebブラウザでありながら、iOSの制約によりAppleのWebKitエンジンを用いて動作するアプリです。そのため表示や動作のコア部分はSafariと共通ですが、Googleアカウントとの連携やUIなど独自の機能が加えられています。iOS版Chromeを利用することで、ブックマークや履歴、パスワードなどをAndroid・PC版Chromeと同期でき、複数の端末間で一貫したブラウジング体験を得ることができます。また、Google の検索機能やフォーム自動入力などが統合されているため、日常的なWeb利用がよりスムーズになります。一方で、レンダリングエンジンはWebKitのままなので、Android版Chromeのような独自エンジン（Blink）による挙動とは異なる点もあります。このようにiOS版Chromeは、Googleのサービスとの親和性を提供しつつ、Appleのプラットフォームに適応した形で動作するブラウザとなっています。
