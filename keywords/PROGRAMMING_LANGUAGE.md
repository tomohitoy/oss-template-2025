# [プログラミング言語]

## 説明

プログラミングとは、簡単に言うとコンピュータにさせたいことを順番に組み立てていく作業のことです。その内容について記述するためにプログラミング言語が使用されています。

コンピュータは「０」と「１」の機械語しか理解できないですが、一方で人間にはそういった機械語は理解できません。そこで、人間が理解できるプログラミング言語を用いて指示を作成し、コンパイラという技術で機械語に翻訳します。

プログラミング言語は様々な種類があり、それぞれの言語によって得意分野が違います。例えば、拓殖大学で一番初めに習ったJavaはアプリケーション開発やバックエンド開発などが得意と言われています。Pythonはデータ分析や機械学習などが得意と言われています。

言語の選択には得意不得意といった要因だけでなく、習得の難易度や一緒に使うべきものがあったりします。例えば、JavaScriptでWebサイトを作成する場合はHTMLやCSSの知識があった方が良いです。三年生の就活をしている方は聞いたことあるかもしれないですが、Rubyという言語には基本的にRuby on Railsといった技術も一緒に使われる場合が多いようです。

最後にYahoo知恵袋によると、開発経験10年のITエンジニアは10個前後の言語の開発経験があるようです。
