# コピーレフトライセンス

## 説明
コピーレフトライセンスとは、プログラムを自由とし、そのプログラムの改変ないし拡張されたバージョンも全て自由であることを要求するための、一般的な手法の一つです。なお、自由とは無償であることではなく、フリーな普及を意味しています。

あるプログラムを自由なソフトウェアにする最も簡単な方法はパプリックドメインにすることです。
しかし、パプリックドメインでは自由に非協力的な人々がプログラムをプロプライエタリソフトウェアにすることも認めることになります。

そこで、ソフトウェアを再配布する人は、変更してもしなくても、それをコピーし変更する自由を一緒に渡さなければならないというコピーレフトライセンスが誕生しました。

コピーレフトライセンスの例としては、GNU GPLが存在します。
GNU GPLでは、再頒布時にソースコードの開示や入手方法の明確化、ライセンス文言の表記、派生物に対するライセンスなど、複数の条件を課しています。
