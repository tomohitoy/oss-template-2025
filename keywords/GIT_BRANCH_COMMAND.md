# git branch コマンド

## 説明

git branch は、Gitでブランチを「見る・作る・消す」ためのコマンドです。

まず、git branch と入力すると、今あるローカルブランチの一覧が表示されます。今自分がいるブランチには * (アスタリスク)がついて分かりやすく示されます。

git branch -a を使えば、リモートブランチも含めた一覧を見ることができます。

新しいブランチを作りたいときは git branch ブランチ名と書くだけです。
ただし、この時点ではブランチを作っただけで、まだそのブランチには移動していません。

作ったブランチに切り替えたい場合は git switch ブランチ名 を使います。

不要になったブランチは git branch -d ブランチ名 で削除できます。

また、ブランチの名前を変えたいときは git branch -m 古い名前 新しい名前 を使います。
