# TypeScript

## 説明

TypeScript（タイプスクリプト）は、Microsoft によって開発され、オープンソースで公開されているプログラミング言語だそうです。

まず「TypeScript とは何か」については、これは JavaScript の「スーパーセット（上位互換）」と呼ばれる言語であると説明されていました。JavaScript のコードはそのまま TypeScript として扱えるようで、JavaScript との互換性を保ちつつ、独自の機能を追加しているとのことです。

その最大の特徴が「静的型付け」という機能のようです。

「何ができるのか」についてですが、JavaScript はプログラムを実行するまでエラーに気づきにくい場合があるのに対し、TypeScript ではこの「静的型付け」によって、コードを書いている段階（コンパイル時）でエラーを検査できるそうです。

これにより、開発者はバグを早期に発見でき、特に大規模なアプリケーション開発において、コードの品質や保守性を向上させ、安心して開発を進められるようになるとのことでした。また、エディタ（開発ツール）でのコード補完やリアルタイムチェック機能が強力になるため、開発効率の向上にも大きく寄与するようです。
