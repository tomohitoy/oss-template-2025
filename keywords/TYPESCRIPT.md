# TypeScript

## 説明

TypeScriptとは、JavaScriptを拡張して開発された新しいプログラミング言語のことだ。JavaScriptとの互換性が高く、大規模開発においてエラーが起きにくい。開発したのはMicrosoft社で、JavaScriptの上位互換として開発された。JavaScriptの機能を維持しながら、より優れた機能を備えている、静的型付け言語だ。また、JavaScripとと違って、TypeScriptにはインターフェイスが搭載されている。そんな、いいことづくしのTypeScriptだが、利用者が少ないのと、習得時間の長さといった、デメリットが存在する。
