# Django

## 説明

Django（ジャンゴ）は、Python で書かれた高水準なウェブフレームワークだそうです。このフレームワークは、経験豊富な開発者によって作られ、安全で保守可能なウェブサイトを迅速に開発することを目指しているとのことです。

Django は「電池入り」の哲学に従っていると説明されており、開発者が一般的に必要とする機能のほとんど（ユーザー認証、管理画面、ORM など）が最初から提供されているようです。これにより、開発者は一般的に必要な機能を自ら開発することなく、アプリケーション本体の開発に集中できるそうです。

また、セキュリティも重視しており、SQL インジェクション、クロスサイトスクリプティング（XSS）、クロスサイトリクエストフォージェリ（CSRF）といった一般的な攻撃に対する保護機能が標準で組み込まれているようです。

アーキテクチャはコンポーネントベースで、拡張性が非常に高く、実際に、Instagram や Disqus のような大規模でトラフィックの多いサイトでも使用されている実績があるとのことでした。
