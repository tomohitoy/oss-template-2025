# [URL (Uniform Resource Locator)]

## 説明

URLとは、インターネット上の特定のウェブページや画像、動画などのリソースがどこにあるかを示す「住所」のような文字列である。例えば「https://www.example.com/page.html」のように、「スキーム（http://やhttps://）」「ドメイン名（www.example.com）」「パス（/page.html）」などの要素から成り立っている。この構造によって、ユーザーはブラウザのアドレスバーにURLを入力するだけで、世界中どこにあっても目的の情報を取得できる。さらに、URLはハイパーリンクの形でWebページ同士を結びつけ、インターネット上の情報の網の目を支えている。 また、URLの設計は検索エンジン最適化にも影響を持ち、「意味のある構造」「階層を反映したパス」「適切なキーワードの使用」が推奨される。
