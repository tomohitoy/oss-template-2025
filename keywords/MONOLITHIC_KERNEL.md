# モノリシックカーネル

## 説明

モノリシックカーネルは、OSの中心部分であるカーネルに必要な機能をすべてまとめて組み込む方式です。
デバイスドライバやファイルシステム、ネットワークなどの機能が同じメモリ空間で動作するため、処理のやり取りが少なく高い性能を出しやすいという利点があります。

しかし、一体化している分、1つの不具合がカーネル全体に影響し、システムが停止してしまう危険もあります。
これに対してマイクロカーネルは必要最小限だけをカーネルに残し、他の機能を外部へ分離する設計です。

現在はその中間であるハイブリッドカーネルを採用するOSも多く、WindowsやmacOSがその例として知られています。Linuxは今もモノリシックカーネル方式を基本としています。
