# [インターネット層]

## 説明

[インターネット層は、TCP/IPモデルの第2層にあたり、異なるネットワーク間でデータパケットを宛先まで届けるためのルーティング機能を担う、通信の土台となる極めて重要な層です。中心となるプロトコルはIPで、データにIPアドレスを付与し、そのアドレスを基に送信元から宛先までの最適な経路を決定し、パケットを転送します。これにより、地理的に離れたネットワークにあるコンピューター同士が通信可能で世界中のネットワーク接続を実現しています。]
