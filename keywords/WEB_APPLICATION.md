# [Webアプリケーション]

## 説明

WEBアプリケーションとは、Webブラウザを介して利用するソフトウェアやサービスです。PCへのインストールが不要で、インターネット経由でサーバー側の処理を利用するため、OSやデバイスを問わず、常に最新機能にアクセスできます。Gmail、Amazon、SNSなどが代表例です。

従来のクライアント/サーバーシステムの課題を解決するために発展し、ブラウザという汎用クライアントを利用することで、サーバー側での一元管理・更新を実現しました。技術的には、プレゼンテーション層（ブラウザ）、アプリケーション層（処理）、データ層（データベース）の「3層アーキテクチャ」が一般的に採用されています。
