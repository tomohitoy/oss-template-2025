# UBUNTU
## 説明

UbuntuはDebian GNU/Linuxを母体としたオペレーティングシステムである。カノニカルという企業から支援を受けており、開発目標は「誰にでも使いやすい最新かつ安定したOS」を提供することとしている。無償で提供されており、ライセンス料も全く発生しない。日本語にも対応していて、インストールせずとも利用が可能である。フリーソフトの開発や、サーバー用途、開発プラットフォームなどなど、用途は多岐にわたる。
