# [秘密鍵]

## 説明

[秘密鍵とは、対になる公開鍵で暗号化された通信を復号化するために使うキーのことで、特定のユーザのみが持つため、公開鍵によって暗号化された電子メールなどの通信は、第三者に読み取られる心配がないため、通信の機密性を保ちながら本人確認の役割も果たすことができます。

公開鍵から秘密鍵を数学的に特定することは現実的に不可能と言われていて、イメージは宝箱を開けられる唯一の鍵を特定のユーザだけが持っているという状態です。ただし、将来的には、十分な性能を持った量子コンピュータの登場で破られる可能性があります。]
