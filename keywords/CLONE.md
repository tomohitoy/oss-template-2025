# [Clone (クローン)]

## 説明

[クローン（Clone）とは、主にGitのようなバージョン管理システムでリモートリポジトリ）の内容を、自分のコンピュータ上に完全に複製すること、またはその複製されたリポジトリ自体を指すことローンを作成することで、開発者はインターネットに接続していなくてもローカル環境でソースコードの閲覧や編集、履歴の確認などが可能になる英単語としての “clone” は「複製する」「模造品を作る」といった意味を持ち、一般的には生物学分野でDNAや細胞の複製を指す言葉としても用いられる]
