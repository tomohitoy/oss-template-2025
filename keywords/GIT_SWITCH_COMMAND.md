# git switch コマンド

## 説明
Gitのswitchコマンドは、ブランチを切り替えるための新しいコマンドで、Gitバージョン2.23から登場しました。以前はcheckoutコマンドでもブランチの切り替えができましたが、checkoutは多機能で少し混乱を招くことがあったため、ブランチ切り替え専用のswitchが作られました。このような背景から、将来的にはcheckoutが使われなくなる可能性もあるらしい。
さらに、同じバージョンで、ファイルの変更を取り消すためのrestoreコマンドも新たに導入されている。
