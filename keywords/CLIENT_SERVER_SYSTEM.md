# クライアント／サーバシステム

## 説明

クライアント／サーバシステムとは、サービスを提供する「サーバ」と、そのサービスを要求する「クライアント」が役割を分担して動作する仕組みのことである。ユーザの操作はまずクライアントが受け取り、表示するページや必要なデータをサーバへリクエストとして送信する。サーバはその要求内容を処理し、結果をレスポンスとしてクライアントへ返す。クライアントは受け取った情報を画面に表示し、ユーザはWebページを閲覧できる。これは処理を分散できるため負荷に強く、故障箇所の特定が簡単で、システムの保守性や拡張性にも優れている点が特徴。
