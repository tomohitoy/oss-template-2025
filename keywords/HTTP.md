# [HTTP (Hypertext Transfer Protocol)]

## 説明

HTTP（HyperText Transfer Protocol）は，「ハイパーテキスト転送プロトコル」の略称であり，主にWebブラウザとWebサーバとの間でHTML文書などの情報をやり取りするための通信手順である。この仕組みによって，クライアントが“GET”や“PUT”などのコマンドをサーバに送信し，サーバからHTMLやJPEG等のデータが返され，ブラウザがそれらを整形・表示することでWebページが成り立っている。HTTPの特徴として，「ステートレス」という性質がある。つまり，1回の要求‐応答が終わるとその後の状態をプロトコル自身が保持せず，次のやり取りは前回の通信を基に自動で継続しない。また，HTTPは通信内容を暗号化していないため，第三者による傍受のリスクがある。個人情報や認証情報を扱う場面では，暗号化された通信方式である HTTPS を用いることが推奨されている。
