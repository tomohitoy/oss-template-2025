
# git add コマンド

## 説明

git add コマンドは、作業ツリーで見つかった現在のコンテンツを使用してインデックスを更新し、次のコミットのためにステージングされたコンテンツを準備する。

通常では、既存パスの現在のコンテンツ全体を追加するが、一部のオプションでは作業ツリーファイルに加えられた変更の一部のみを適用してコンテンツを追加したり、既に作業ツリーに存在しないパスを削除したりすることも出来る。

git add コマンドは作業ディレクトリからリポジトリにファイルを圧縮し、blobオブジェクトとして40文字のhash値を名前として格納するという処理と、その40文字のhash値をステージングエリアのindexファイルに追加するという処理をしている。
