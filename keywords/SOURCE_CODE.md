# ソースコード

## 説明

プログラミング言語で書かれたコンピュータプログラムを表現する文字列のことである。
ソースコードはプログラミング言語を用いて書かれるものであり、コンピュータハードウェアのための機械語よりも人間が読み書きするのに適している。
設計から実装へ、というプログラミングの過程でソースコードを記述することを特に指してコーディングという。
ソースコードの量は通例ソフトウェアの規模に比例して増大する。大規模なソフトウェアの場合はソースコードを分類することでメンテナンス性を確保する。ソースコードの行数をもとにソフトウェア開発プロジェクトの規模を見積もる手法も存在する。
