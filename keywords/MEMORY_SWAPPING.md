# メモリースワッピング

## 説明
メモリースワッピングとは、主記憶が不足した際に, 一部のデータをスワップ領域に一時的に退避させる仕組みのことである.

使わないデータを退避させることを”スワップアウト”, 退避したデータをメモリに戻すことを"スワップイン"という。
OSは使用頻度や優先度に応じてスワップするデータを選び, メモリの空き領域を確保している.

メモリースワッピングの仕組みによって, 物理メモリ以上のアドレス空間を仮想的に作ることができ, 
複数の処理を同時かつ安全に実行可能になる.
