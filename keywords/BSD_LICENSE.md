# [BSDライセンス]

## 説明

BSDライセンスとは、BSD系OSから生まれた非常に緩やかなオープンソースライセンス。
特徴は「利用者にほとんど制限を課さない」ことにあり、誰でも自由に 利用・改変・再配布・商用利用 ができます。さらに、MITライセンスと同様に「著作権表示」と「無保証の明記」を残すだけでよく、改変したソフトをクローズドソースのまま販売することも許可されています。この柔軟性の高さから、企業製品やプロプライエタリソフトにも広く取り込まれ、macOS や PlayStation のネットワークスタックなど、多くの商用製品に影響を与えてきました。
