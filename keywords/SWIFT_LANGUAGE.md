# Swift言語

## 説明

Swift言語とは、2014年にApple社が作った新しい言語です。この言語は、iOSやMac向けのネイティブアプリケーションを開発するために開発されました。ハードウェアとソフトウェアの両方をAppleが制作しているため、Swiftならディスクやメモリなど、コンピュータの深い部分やハードウェアに関わるアプリを作ることができます。特徴は、現代的なモダンプログラミング、他の言語との相互性、名前の通り「迅速」なパフォーマンス、Playgroundの実行環境があげられます。活用するメリットは、処理速度が速い、不具合が起きにくい、コードが書きやすく読みやすい、他の言語との互換性良い、Playgroundにより修正箇所を見つけやすい、初心者でも書きやすいといったものがあげられます。また、Swift言語によってできることは、iPhone、iPad向けアプリの開発、MacBook、iMac向けアプリの開発、Apple TV向けアプリの開発、Apple Watch向けアプリの開発、Webアプリケーション／Webサーバーの構築の構築ができるようになっています。
