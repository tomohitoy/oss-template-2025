# ChromeOS

## 説明

ChromeOSは、Googleによって開発された、Linuxカーネルをベースとするオペレーティングシステムです。
主にChromebookに搭載されており、その最大の特徴は、システム全体がWebブラウザであるGoogle Chromeを中心に設計されている点にあります。ほとんどの処理やアプリケーションの実行は、クラウド上で行われることが前提であり、インターネット接続環境での利用に特化しています。このような設計思想によって、OS自体が非常に軽量で起動が速く、リソースの少ないハードウェアでも快適に動作します。
