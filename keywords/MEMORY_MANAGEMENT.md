# メモリ管理

## 説明
メモリ管理とは, プログラムが動作するために使うメモリ領域を割り当て, 解放し, 
プログラムが安全かつ効率よく動作するように管理する仕組みである.

メモリの割り当ては, プログラムがデータを保存するために必要なサイズのメモリを使えるようにするために, メモリ上の領域を確保することを指す.
また, メモリの解放は, メモリを無駄に使い続けないように使い終わったメモリ領域を空けることを指す. 

これらを適切に管理し, 限られたリソースを効率的に使ってプログラムを安定して動かすのがメモリ管理の役割である.
