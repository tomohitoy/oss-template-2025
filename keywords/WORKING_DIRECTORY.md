# [作業ディレクトリ (Working Directory)]

## 説明

[「作業ディレクトリ（working directory）」とはユーザーが現在ファイル操作などを行っているディレクトリのことを指す。また「カレントディレクトリー」という言葉と同じ意味を持っている。具体的には、シェルやファイルエクスプローラ上でコマンド実行やファイル参照が行われる「基準地点（出発点）」としての役割を持つ。作業ディレクトリは、cd（change directory）コマンドを使って移動・変更ができ、移動後はそのディレクトリを起点にして相対パスでファイルを指定できる。さらに、現在の作業ディレクトリを確認するには pwd コマンドを用いることで、その出力結果として絶対パスが表示される。]
