# パブリックドメイン (PD)

## 説明

「パブリックドメイン（Public Domain）」とは、著作物・発明・意匠などの知的創作物について、著作権その他の知的財産権がもはや存在しない、あるいはそもそも発生しなかった状態を指します。日本語では「公有」「公共領域に帰した」と訳されることもあります。
具体的には、たとえば日本では著作者の没後70年が経過した著作物は著作権の保護期間が満了し、パブリックドメインに入るとされています。また、著作者が明示的に著作権を放棄したり、著作物として保護要件を満たさなかったり、そもそも保護対象外と法で定められていたりする場合にも、当該作品・発明はパブリックドメインになるとされます。
パブリックドメインに一旦入った著作物は、誰でも自由に複製・翻案・配布・表示などを行えるようになるため、例えば教育資料・翻訳・再出版・創作活動の素材として有効に活用できます。ただし、完全に「なんでも好きにできる」わけではなく、著作者の「人格的利益」は著作権の保護期間消滅後も一定の配慮が必要、という解釈もあります。
このように、パブリックドメインの作品を適切に活用することで、創作や教育、ビジネスの幅が広がる反面、対象の保護期間・国・法域を確認したり、「著作権フリー」や「ロイヤリティフリー」との違いを理解したりする慎重さも求められます。
