# [OS (オペレーティングシステム)]

## 説明
OS（オペレーティングシステム）とは、コンピュータのハードウェアとソフトウェアの橋渡しを行う基本的なシステムソフトウェアのことを指す。ユーザーがアプリケーションを利用できるように、CPUやメモリ、ストレージ、入力装置などのシステムリソースを管理し、効率的に動作させる役割を持つ。代表的なOSには、Microsoft Windows、macOS、Linux、Android、iOSなどがある。OSは「資源の管理」「ユーザーとのインターフェース」「アプリケーションの実行環境の提供」という3つの主要な機能を担っており、現代のコンピュータやスマートフォンの動作に欠かせない基盤である。
