# コミットハッシュ

## 説明

コミットハッシュは、Gitでの各コミットを一意に識別するための英数字文字列で、このハッシュを使って特定のコミットを参照したり、操作したりすることができる。
例えば 8a7d3f1e9b...のような文字列で、これを使って特定のコミットに戻ったり、参照したりできる。
コマンドハッシュには短縮ハッシュとフルハッシュと呼ばれる文字数の違う2種類が存在する。コミット数が多くなければ大抵は短縮ハッシュと呼ばれる8桁ほどの文字列で問題なく動作できる。
