# コンパイル

## 説明

コンパイルとは、指定された言語で書かれたプログラム（ソースコード）を別の形式や言語の一連の命令に変換する作業のことです。
コンパイルはコンパイラで行われます。

多くの場合、コンパイラは高級言語で書かれたコードをバイナリコードや機械語に変換します。
また、全てのソースコードを実行前にコンパイルします。
しかし、実行時にコードを一行ずつ機械語に変換する言語もあります。その言語をインタプリタ言語と言います。

Javaのコンパイルはjavacコマンドで行います。
.javaファイルをコンパイルすることで、.classファイルが作られ、実行することができるようになります。
