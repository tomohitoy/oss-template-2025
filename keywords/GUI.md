# GUI

## 説明

[GUIは、アイコンやボタン、ウィンドウ、メニューなどの視覚的要素を組み合わせ、マウスやタッチ操作で直感的に操作できる仕組みです。コマンド入力を必要とするCUIに比べ初心者に優しく、スマートフォンやパソコンをはじめ家電や自動販売機など幅広く採用されています。利便性を高めるために配色や配置、アクセシビリティ設計が重要で、デザインによって操作性が大きく変わります。初期のパソコンではCUIが主流でしたが、GUIはMacやWindowsの普及で一般化しました。]
