
# WWW(World Wide Web)

## 説明

WWW(World Wide Web)は、1990年代にARPANETを引き継ぎ発展した技術として発表され、学術利用だけでなく商用利用も可能なものとして発展した。

WWWがあることによって、インターネット上に公開されているWebページ同士をリンクさせて表示させることができるようになった。

WWW上には無数のWebサイトが開設されており、企業、団体、学校、個人等がWebサイトにアップロードした情報を、インターネットユーザーは自由に閲覧することが出来る。
