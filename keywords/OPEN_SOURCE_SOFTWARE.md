# [オープンソフトウエア]

## 説明

[オープンソースソフトウエアとはオープンコラボレーションによって開発および維持されるソースコードのことです。ossは誰でも自由に使用、調査、変更、再使用することができ通常は無料で利用できる。ossの具体例として、wordpressやbraveのbrowserが挙げられます。ossを使うメリットは開発にかかる時間と費用の削減や、高いセキュリティ性などが挙げられ、デメリットとしては開発プロジェクトの継続性が挙げられます。有志で成り立っている為プロジェクトが遅延したり突然中止になってしまう可能性がありす。]
