フリーソフトウェア運動（Free Software Movement）は、「コンピュータの利用者がソフトウェアを自由にコントロールできるようにすべきだ」という社会運動や思想のこと。
1983年にリチャード・ストールマン（Richard Stallman）によって提唱され、彼が設立したフリーソフトウェア財団（Free Software Foundation, FSF）が中心的な組織となっている。
具体的には、ソフトウェアの利用者が以下の「4つの基本的な自由」を持つべきだという倫理的・政治的な主張。
1.	プログラムを実行する自由。
2.	プログラムを研究し、改変する自由（ソースコードへのアクセスが前提）。
3.	コピーを再配布する自由。
4.	改良版を公衆に配布する自由。
