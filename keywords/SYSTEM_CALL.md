# [システムコール]

## 説明

[実行中のアプリケーションプログラムがOSの持つ特定の機能を呼び出すもの。
アプリケーションは通常、セキュリティの観点からハードウェアなどに勝手にアクセスすることはできない。しかし実行中にOSの機能が必要になる場合、システムコールを実行することでOSに処理を依頼することができる。この時CPUはカーネルモードに切り替わり処理が完了すると結果がアプリケーションに返される。このシステムによりアプリケーションがシステムの中核を破壊することを防いでいる。
]
