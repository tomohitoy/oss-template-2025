# [公開鍵]

## 説明

[公開鍵とは、通信を暗号化するときに使うキーのことで、公開鍵暗号方式または非対称暗号方式やデジタル署名で使われる２種類の暗号鍵の一つで、誰でも利用できるように広く公開されています。

公開鍵暗号方式では、公開鍵を使用してデータを暗号化し、その暗号化されたデータは対応する秘密鍵を持つものだけが複合できます。これによって、公開鍵を知っている者が情報を安全に送信でき、秘密鍵を持つ者だけがその情報を読むことができます。

また、デジタル署名では、秘密鍵を使用してデータに署名し、その署名は公開鍵を使用して検証されます。これにより、データの完全性(Availability)と送信者の身元を確認することができます。
]
