# シェル (種子の殻)

## 説明

シェルとは、ソフトウェアの一種で、カーネルというソフトウェアとユーザーとの間を取り持つ、インターフェースの役割を果たします。

shellという単語は、日本では一般的には「貝殻」という意味で訳されることが多いです。しかし、karnelが「仁（果実の核）」という意味であることや、karnelとshellの関係性から、shellは、「貝殻」ではなく「種子の殻」という意味に由来していると考える方が自然かもしれません。
