# コミット (Commit)

## 説明

データベースにおける、トランザクション処理において、関連する処理がすべて成功し、結果を確定させることをコミットという。
トランザクション処理では一体不可分な複数の処理が「すべて成功」か「すべて失敗」のいずれかになるよう制御している。すべての処理が成功し、データベースの状態を処理結果によって更新することをコミットという。
簡単にまとめると一連の動作を行い、その結果を確定させることがコミットに当たる。
また、ビジネス用語として使われる場合もある。この場合は約束すること、合意することを示す。
