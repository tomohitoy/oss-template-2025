# [ユーザーモード]

## 説明

[ユーザーモードとは、オペレーティングシステムにおいてアプリケーションが動作する際の実行モードであり、CPU が利用できる命令や資源が制限された状態を指す。ユーザーモードで動くプログラムは、ハードウェアを直接操作することができず、メモリ領域にも強い制限があり、OSの管理する重要領域へ勝手にアクセスすることはできない。この制限によって、アプリケーション同士が互いに干渉することを防ぎ、誤動作によるシステム全体の破壊を避ける仕組みになっている。
アプリケーションがファイル操作や入出力装置の利用などを行う場合、ユーザーモードのままでは実行できないため、システムコールを通じて OS に依頼し、OS がカーネルモードで処理を実行する。つまり、ユーザーモードは安全性を確保するための“制限された領域”であり、カーネルモードと役割を分離することで安定した環境を実現している。]
