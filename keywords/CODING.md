# コーディング

## 説明

コーディングとは、ホームページを表示するために必要な「ソースコード（コンピューターに命令する文章）」を書くことです。
コーディングとよく混同される言葉として、「プログラミング」があります。
コーディングとプログラミングの違いは、2つの視点から考えられ、作業範囲の違い、言語の違いがあります。
作業範囲における違いとして、プログラミングは、コーディングを含めた作業全体のことで、設計、テスト、修正などの作業も含まれています。
つまり、コーディングは、プログラミングにおける作業の一部です。
言語における違いとして、コーディングではHTML、CSS。プログラミングではPython、Javaがメジャーに使用されると考えられます。
とはいえ、Web制作現場やインターネット上の情報など、シーンによってこれらが混同して使われている場合もあるので、あくまでも参考程度です。
