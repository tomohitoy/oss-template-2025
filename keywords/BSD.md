# [BSD]

## 説明

BSD（Berkeley Software Distribution）は、1970～80年代にカリフォルニア大学バークレー校で開発されたUNIX系オペレーティングシステムの系統です。研究者や学生がAT&T UNIXを改良し、高品質なネットワーク機能（TCP/IPスタックなど）を実装したことで、インターネットの発展に大きく貢献しました。BSDの特徴は「BSDライセンス」という非常に自由度の高いライセンスにあり、商用利用や再配布が容易なため、多くのOS（FreeBSD、NetBSD、OpenBSD、macOS など）に影響を与えました。現在でも信頼性・安全性の高いUNIX系OSとして幅広く利用されています。
