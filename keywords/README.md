# キーワードリスト

このディレクトリには、受講生が選んだキーワードとその解説が格納されています。

## 概要

「オープンソースソフトウェア」の講義で学んだ内容の中から、最も印象に残ったキーワードを1つ選び、その解説を記述してください。

## ファイルの作成方法

1. 自分の学籍番号のファイル（例：`B1234567.md`）を作成
2. 以下の形式で記入

```markdown
# [選んだキーワード]

## 説明

[キーワードの解説を100〜300字程度で記入]

## 作成者

- 氏名: [あなたの氏名]
- 学籍番号: [あなたの学籍番号]
```
