# [JavaScript]

## 説明

JavaScriptとは、Webサイトのブラウザ上で動くプログラミング言語の一つです。一般的には、動的なWebページやアプリを開発する際に使われてるというイメージがありましたが、最近では、JavaScript一つとってもできることは多様になりました。元々は、1995年にFireFoxの前身であるNetscapeというブラウザにプログラムを加えるために生まれました。他の言語との違いは、JavaScriptは、フロントエンド言語と呼ばれており、Webブラウザ側で処理が行われます。一方でPHPのような言語は、バックエンド言語と呼ばれており、Webサーバー側で処理が行われることが大きな違いがあります。JavaScriptを用いてできることは、Webサイト上でコンテンツがクリックされたら、コンテンツの色を変えること、ページがスクロールされたら、ポップアップを表示すること、画像のスライドショーを再生すること、Cookieを操作して、Webサイトに訪問したユーザーの情報を取得すること、Webサーバーにフォームのデータを送信すること大きくできることです。JavaScriptの強みは、高い汎用性、HTMLとCSSの二つを組み合わせることで、技術の拡大、開発環境の充実とクロスプラットフォーム対応などがあげられます。よく間違われる言語のJavaとは別物で、JavaはJVMで動くコンパイル言語、JavaScriptはブラウザ等で手軽に動く汎用言語といったように少し違います。
