# [トランスポート層]

## 説明

[トランスポート層は、データの送信元と送信先間における通信制御を担うプロトコル階層で、OSI参照モデルでは第4層、TCP/IP階層モデルでは第3層に位置する。この層の役割としては、エラー検出・訂正、再送制御、コネクションの確立（仮想専用通信路）、データのまちがった順序を整える順序制御、フロー制御、輻輳制御、さらにはアプリケーション識別などがある。代表的なプロトコルとして、信頼性を重視する TCP（Transmission Control Protocol）と、速度を重視し制御機能を省略した UDP（User Datagram Protocol）がある。]
