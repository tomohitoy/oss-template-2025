# HTML (Hyper Text Markup Language)

## 説明

HTML（Hyper Text Markup Language）は、Webページを作成するための基本的な「マークアップ言語」です。
HTMLはページの構造を定義する役割を持ちます。
例えば、文章が「見出し」なのか「段落」なのか、あるいは「リンク」なのかといった役割を、「タグ」と呼ばれる <h1> や <p> のような印を使って指定します。
CSSが見た目の装飾を担当するのに対し、HTMLはWebページ上のテキストや画像などの要素に「どのような意味を持つか」を指示し、骨組みを作るのが主な役割です。
