# NGINX

## 説明

NGINX（エンジンXと呼ばれる）とはHTTP Web サーバー、リバース プロキシ、コンテンツ キャッシュ、ロード バランサー、TCP/UDP プロキシ サーバー、およびメール プロキシ サーバーである。
Igor・Sysoeというカザフスタン人が元の開発者で、現在F5 Networks, Inc.という会社が所有している。
BSDライセンスで手配している。
Nginx は、Apache Web サーバーを上回るパフォーマンスを実現するという明確な目標を持って作成され、その目標が達成していたが、現在は後者の方がより使われているそうである。
